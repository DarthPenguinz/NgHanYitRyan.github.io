FPGA Arcade Game

To help us understand computational structures better, we were tasked as groups to create arcade games using the Alchitry IO Fully Programmable Gate Arrays (FPGA). As a team, we chose to make a game similar to Pinao Tiles, where players must press all allocated buttons before the rows shift. Below we have a video on how the game is played.
